## Project 1
### Query 1
Given a disease id, obtain its correlated information from data.
### Query 2
We assume that a compound can treat a disease if the
compound up-regulates/down-regulates a gene, but the
location down-regulates/up-regulates the gene in an opposite
direction where the disease occurs. Find all compounds that can
treat a new disease (i.e. the missing edges between compound
and disease excluding existing drugs).
